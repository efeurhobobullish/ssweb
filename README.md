# 🌐 SSWebVid

![Node.js](https://img.shields.io/badge/built%20with-Node.js-brightgreen)
![Playwright](https://img.shields.io/badge/uses-Playwright-blue)

> A minimalist API server to take **screenshots** and **video recordings** of websites using Playwright and Express.

---

## ✨ Features

- 📸 Capture full-page **PNG screenshots**
- 🎥 Record **15-second MP4 videos** (720p)
- ⚡ Fast, lightweight, and temporary (no permanent storage)
- 🚀 Simple REST API, ready for deployment

---

## 🛠 Tech Stack

- **Node.js** (Express)
- **Playwright** (Chromium)
- Native **FS & Path modules**

---

## 📦 Install & Run

```bash
npm install & npm start
